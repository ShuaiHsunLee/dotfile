## Installation
### macOS:
```
brew install tmux
```
### Ubuntu:
```
source install_tmux
```

## Customization
```
cp .tmux.conf ~
```

## Preferred Plugin
[tmux-plugins/tpm](https://github.com/tmux-plugins/tpm)<br>
[tmux-plugins/tmux-prefix-highlight](https://github.com/tmux-plugins/tmux-prefix-highlight)<br>
