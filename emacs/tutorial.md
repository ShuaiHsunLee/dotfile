# Emacs

## Basic tutorial
M-x mode-name

close emacs
```
C-x Cc
```

open file
```
C-x C-f
```

save file
```
C-x C-s
```

Switching between buffers
```
C-x b and C-x C-b
```

Cut, paste
```
C-k, C-y
```

Keyword search
```
C-s and C-r
```

Undo
```
C-/
```

Invoking commands by name
```
M-x name
```
